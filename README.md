# Telegram Bot: FilmButler

## Опис проекту
**FilmButler** — це Telegram бот, що дозволяє користувачам:
- Знаходити фільми за назвою.
- Отримувати список актуальних фільмів у трендах.
- Одержувати рекомендації на основі обраного фільму.
- Отримувати посилання на трейлер фільму разом із інформацією про нього.

Бот використовує [The Movie Database (TMDB)](https://www.themoviedb.org/) API для пошуку та обробки інформації про фільми. Уся інформація надається українською мовою, якщо це можливо.

## Інструкція для запуску
### Необхідне програмне забезпечення
**Python** (версія 3.10 або вище).  

### Встановлення необхідних бібліотек
pip install aiogram aiohttp

### Запуск бота
python main.py

### Після цього у Telegram відкрийте бота за юзернеймом @FilmButler_bot та в чаті напишіть команду /start
